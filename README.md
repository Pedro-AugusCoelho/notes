# Notas

Projeto criado com Expo (TypeScript) e estilização com styled-components.

## Requisitos
- Node.js 20+ recomendado
- Expo CLI via `npx`

## Instalação

```bash
npm install
```

## Executar

```bash
npm run start
# ou
npm run android
npm run ios
npm run web
```

## Tecnologias
- Expo
- React Native
- TypeScript
- styled-components

## Estrutura
- [App.tsx](App.tsx): exemplo mínimo usando styled-components.
